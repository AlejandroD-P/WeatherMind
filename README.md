# WeatherMind

WeatherMind es una aplicación de escritorio desarrollada en Python que consulta información meteorológica y genera recomendaciones para actividades al aire libre.

## Estado del proyecto

Versión actual: v0.3

## Funcionalidades actuales

- Consulta de clima mediante API externa.
- Selección de actividad.
- Cálculo básico de riesgo.
- Generación de recomendaciones.
- Interfaz gráfica con CustomTkinter.

## Tecnologías

- Python
- CustomTkinter
- Requests
- Open-Meteo API

## Próximas mejoras

- Dashboard visual por tarjetas.
- Historial con SQLite.
- Gráficas de pronóstico.
- Módulo de IA para recomendaciones.
